# Leachate Prediction App

This Streamlit application predicts leachate formation based on rock characteristics and a sequence of environmental events (Sequence S). It is designed to be **user-friendly**, even for non-technical users.

---

## Features

- Input **custom rock characteristics** or select from predefined rocks:
  - Basalt
  - Granite
  - Limestone
- Enter multiple **environmental events** in Sequence S:
  - Precipitation type (`rain` or `snow`)
  - Acidity (numeric, e.g., 0.6)
  - Temperature after the event (numeric, e.g., 12)
